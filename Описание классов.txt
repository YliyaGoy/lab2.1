# Комментарии к коду:

1. **Соглашения по именованию полей и методов:**
   - Поля и методы, начинающиеся с одного подчеркивания (_), считаются защищенными (protected) и предназначены для внутреннего использования в классе.
   - Поля или методы, начинающиеся с двух подчеркиваний (__), считаются приватными (private).

2. **Класс Room:**
   - Содержит методы для установки и получения площади, высоты и стиля комнаты.
   - Присутствует приватный метод `__private_method`, который вычисляет и выводит периметр комнаты.
   - Публичный метод `public_method` вызывает приватный метод `__private_method`.

3. **Класс OneRoomFlat:**
   - Содержит информацию о комнате, площади кухни и этаже.
   - Методы для установки и получения данных о квартире.

4. **Класс PublicOneRoomFlat:**
   - Наследуется от OneRoomFlat и добавляет информацию о городе.
   - Публичный метод для вывода деталей о квартире.

5. **Класс _StudioApartment:**
   - Содержит информацию об апартаментах, включая студию.
   - Методы для установки и получения данных о студии.

7. **Методы room_menu, studio_menu, flat_menu:**
   - Содержат функционал для работы с комнатой, студией и квартирой соответственно.

8. **Метод main:**
   - Создает экземпляры классов и запускает основной цикл программы.
